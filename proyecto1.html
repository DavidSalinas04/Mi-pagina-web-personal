Portafolio

Proyecto 1: Creacion de pagina web y videojuego

Este proyecto consiste en el diseno de una pagina web para el la empreas de rockwell automatization, tiene funciones basicas como el inicio de sesion, registro de usuarios, y un catalogo de productos, y el apartado del juego creado en unity.

Funcionalidades de la pagina web

  • Inicio de sesion
  • Registro de usuarios
  • Catalogo de productos

Especificaciones

  • Frontend: React
  • Backend: Node.js
  • Base de datos: PostgreSQL

Funcionalidades del videojuego

  • Mapa de diferentes niveles
  • Incremento de puntaje
  • Modelado de figuras en 3D
