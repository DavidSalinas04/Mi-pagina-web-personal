Portafolio

Desarollo de tren automatizado

Este proyecto de Internet de las Cosas (IoT) tiene como objetivo la automatización de un sistema utilizando diversos sensores y actuadores, con la capacidad de recopilar, procesar y visualizar datos en tiempo real. Mediante el uso de un microcontrolador Arduino, se integraron varios sensores (como temperatura, movimiento, Rfid, y de luz.) y actuadores (como servomotores, ventiladores, y leds.) para monitorear y controlar distintos parámetros en el entorno físico.

El sistema está diseñado para operar de manera automática, tomando decisiones en función de los datos recolectados por los sensores. Por ejemplo, si el sensor de temperatura detecta que la temperatura es demasiado baja, el sistema activa un ventilador a través de un actuador. Además, en función de la información de otros sensores (como de luz o movimiento), el sistema puede realizar ajustes automáticamente.

Para garantizar la eficiencia y el control remoto del sistema, los datos recolectados por los sensores se envían a una base de datos utilizando APIs desarrolladas específicamente para este fin. Esto permite que los datos sean accesibles desde cualquier lugar y en cualquier momento.

La visualización de los datos se realiza a través de Power BI, donde se crean gráficos e informes interactivos que permiten analizar los datos históricos y en tiempo real. De esta forma, el usuario puede visualizar y tomar decisiones basadas en los datos recolectados, optimizando el rendimiento y mejorando la automatización del sistema.
